# Support

**Fist Strike VR** support is provided by the Fist Strike team. Support requests can be sent to:

**Email:** wazi1010+fsvr@gmail.com

Please include the following information when contacting support, if applicable:

- App name: Fist Strike VR
- Device model, such as Meta Quest 2, Meta Quest 3, or Meta Quest Pro
- App version, if visible in the App or store listing
- A short description of the issue
- Steps to reproduce the issue
- Screenshots, video, or error messages, if available

We will review support requests and respond as soon as reasonably possible.

## Privacy and Data Requests

Fist Strike VR stores gameplay settings, progress, and training statistics locally on the device. To delete local App data, clear the App's data from Meta Quest system settings or uninstall the App.

For privacy questions or data deletion assistance, contact:

**wazi1010+fsvr@gmail.com**

## Safety Notice

Fist Strike VR is a virtual reality boxing and fitness application. Use the App in a safe play area, take breaks as needed, and stop using the App if you feel discomfort, pain, dizziness, or other unusual symptoms.
